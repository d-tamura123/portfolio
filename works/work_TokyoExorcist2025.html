Tokyo Exorcist 2025

ホラーオカルト × エクソシスト視点アクションRPG

概要

本作はホラーオカルトを題材にした、エクソシスト視点で異界を探索するアクションRPGです。
不気味な雰囲気と探索要素を重視し、霊視や疑似3Dサウンドによって「オカルト世界を体験する感覚」を強調しています。

動作環境

  • OS: Windows 10 / 11
  • CPU: Intel Core i5 相当以上推奨
  • メモリ: 4GB以上
  • グラフィック: DirectX 11 対応
  • 入力: ゲームパッド必須（Xboxコントローラ基準）

ゲーム映像

技術的ポイント

  • ゲーム基盤の自作: シーン管理・入力レイヤー・イベント・ミッション・マップなどを独自フレームワークとして構築。
  • データ駆動設計: キャラ配置・ミッション進行・イベントを JSON / CSV で管理し、マスターデータからゲーム内へロード。
  • 外部ツール連携: Tiled Editor ＋ Tileson を導入し、マップデータを外部化して工数削減とレベルデザインの柔軟性を確保。
  • 疑似3Dサウンド: 距離減衰＋方向性SE＋BGMフェードにより、敵接近や緊張感を音響で表現。
  • 霊視システム: 右トリガー入力で霊が可視化される「霊視」を実装し、視覚・聴覚の両面からオカルト体験を強化。
  • ミッション／イベント連携: ミッション進行に応じて NPC・モンスター・イベントを切り替える仕組みを構築し、シナリオ進行を制御。
  • SpriteTool（自作ツール）: Unity の 3D キャラクターをスプライトシート画像に変換するための 自作ツールを開発し、アニメーション素材の制作工数を大幅に削減。 GitHub リポジトリ

使用技術（Tech Stack）

  • C++17
  • DXLib（DirectX 11）

制作領域（Scope）

  • ゲーム基盤の構築： シーン管理、入力レイヤー、イベント、ミッション、マップ管理など ゲーム全体のロジックと仕組みを自作。
  • データ駆動設計： JSON / CSV によるマスターデータ管理、ロード処理の実装。
  • マップ／レベルデザイン： Tiled Editor を用いたマップ制作、Tileson によるデータ連携。
  • 音響システム： 疑似3Dサウンド、距離減衰、方向性SE、BGMフェードの実装。
  • UI／演出： メニュー、ステータス表示などの基本的な UI 実装。
  • アート／素材管理： フリー素材の選定・加工・統合。
